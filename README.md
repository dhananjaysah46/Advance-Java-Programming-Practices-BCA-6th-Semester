# Advance Java Programming Practices (BCA 6th Semester)

This repository serves as a collection of advanced Java development exercises, focusing on enterprise-level concepts, web applications, and sophisticated GUI designs for the BCA 6th Semester curriculum.

## 📌 Overview
The projects here transition from standard Java to **Java EE (Enterprise Edition)** and advanced desktop components, covering the core pillars of modern Java applications.

## 📂 Project Directory

| Resource Name | Description |
|:--- |:--- |
| **ServletPractice / .zip** | Server-side programming using Java Servlets to handle HTTP requests. |
| **MVCProject.zip** | Implementation of the Model-View-Controller design pattern. |
| **JavaBeanExample.zip** | Working with reusable software components (JavaBeans). |
| **GridBagLayoutTest.zip** | Advanced Swing GUI design using the GridBagLayout manager. |
| **Menus.java.zip** | Implementing MenuBars, Menus, and MenuItems in desktop apps. |
| **advancejava** | Collection of miscellaneous advanced Java source files. |

## 🛠️ Key Topics Covered
* **Java Servlets:** Life cycle, Request/Response handling, and Session management.
* **MVC Architecture:** Separation of business logic, data, and presentation.
* **JavaBeans:** Encapsulation, serializability, and zero-argument constructors.
* **Advanced Swing:** Complex layouts and event handling for desktop applications.
